fillRect

The Official Description

The CanvasRenderingContext2D.fillRect() method of the Canvas 2D API draws a filled rectangle at (x, y) position whose size is determined by width and height and whose style is determined by the fillStyle attribute.

Mozilla Developer's Network

Draw A Rectangle and Fill It In

context.fillRect() draws a rectangle and fills it in with the color defined by context.fillStyle.

Editor (write code below)
var canvas = document.getElementById('fill_rect_interactive1'); var context = canvas.getContext("2d"); context.clearRect(0, 0, 450, 300); context.fillStyle = 'red'; context.fillRect(100,100,100,50);
