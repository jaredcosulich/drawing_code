Math.random()

The Official Description

The Math.random() function returns a floating-point, pseudo-random number in the range [0, 1) that is, from 0 (inclusive) up to but not including 1 (exclusive), which you can then scale to your desired range. The implementation selects the initial seed to the random number generation algorithm; it cannot be chosen or reset by the user.

Mozilla Developer's Network

Using Math.random()

You will often use Math.random() along with a scalar and then round the number to get a random integer.

The following example creates between 1 and 10 black squares randomly distributed around the canvas. Pressing "Run" multiple times will demonstrate the randomness.

Editor (write code below)
var canvas = document.getElementById('random_interactive1'); var context = canvas.getContext("2d"); context.clearRect(0, 0, 600, 360); var squareCount = Math.ceil(Math.random() * 10) for (var i = 0; i < squareCount; ++i) { var x = Math.floor(Math.random() * 350); var y = Math.floor(Math.random() * 250); context.fillRect(x,y,50,50); }
