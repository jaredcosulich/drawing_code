lineTo

The Official Description

The CanvasRenderingContext2D.lineTo() method of the Canvas 2D API connects the last point in the sub-path to the x, y coordinates with a straight line (but does not actually draw it).

Mozilla Developer's Network

Draw A Line (but not actually draw it?)

context.lineTo() creates the path for a line, but won't draw anything.

You'll need to call context.stroke() to actually draw the line.

Editor (write code below)
var canvas = document.getElementById('line_to_interactive1'); var context = canvas.getContext("2d"); context.clearRect(0, 0, 450, 300); context.beginPath(); context.moveTo(100, 250); context.lineTo(400, 50); context.stroke();
