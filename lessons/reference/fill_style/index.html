The Official Description

The CanvasRenderingContext2D.fillStyle property of the Canvas 2D API specifies the color or style to use inside shapes. The default is #000 (black).

Mozilla Developer's Network

Different ways to describe colors.

context.fillStyle allows you to set the color or gradient for a rectangle drawn with context.fillRect(). You can set the color in different ways:

  • Named Colors
    fillStyle will accept named colors (e.g. 'red'). Here is a full list of HTML5 color names
  • Hex Values
    Colors can also be described with a hex value (e.g. #00FF00 is red). You can find hex values for different colors at color-hex.com
  • RGB Values
    Colors can also be described by an rgb value (e.g. rgb(255,0,0) is red). The three numbers in the "rgb()" method describe how much red, green, and blue are present in the color with 0 = none and 255 = 100%. You can find RGB values for different colors using this tool: RGB Colors

Example

You can also create gradients using fillStyle.

Editor (write code below)
var canvas = document.getElementById('fill_style_example1'); var context = canvas.getContext("2d"); var gradient = context.createLinearGradient(0,0,200,0); gradient.addColorStop(0,"green"); gradient.addColorStop(0.5,"white"); gradient.addColorStop(1,"blue"); context.fillStyle = gradient; context.fillRect(10,10,200,100);
